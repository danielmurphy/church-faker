# ChurchFaker

A module for the fantastic [ffaker gem](https://rubygems.org/gems/ffaker) to generate random church names.

## Installation

Add this line to your application's Gemfile:

    gem 'church_faker'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install church_faker

## Usage

    Faker::Church.name
